Minimum Hardware Requirements

To run the IQ World digital signage software effectively, your device must meet the following minimum hardware specifications.

Operating System

The IQ World platform requires an Android-based operating system. Devices must run a supported version of Android to ensure compatibility with platform updates and security patches.

RAM Requirements

A minimum of 1 GB RAM is required for standard signage operation. For optimal performance, 2 GB RAM or more is recommended.

  • 1 GB RAM minimum for basic content playback.
  • 2 GB RAM recommended for multitasking and 4K content.
  • Higher RAM improves responsiveness and reduces buffering.

Storage Requirements

A minimum of 8 GB internal storage is required for the operating system, IQ World software, and cached content.

  • 8 GB internal storage minimum.
  • 16 GB or more recommended for extensive media libraries.
  • External USB storage supported for additional capacity.

Connectivity Requirements

Reliable network connectivity is essential for cloud-based management, content updates, and remote monitoring.

  • Wi-Fi 802.11 b/g/n/ac for wireless setups.
  • Ethernet (RJ-45) for wired installations requiring maximum stability.
  • DHCP or static IP configuration supported.
  • Proxy and firewall support available.

Network Speed

A minimum network speed of 10 Mbps is required for reliable platform operation.

  • 10 Mbps minimum for standard content synchronization.
  • 25 Mbps or higher recommended for 4K content and large media files.
  • Stable low-latency connection improves real-time control.

USB Support

USB ports enable direct content upload, peripheral connectivity, and firmware updates.

  • USB 2.0 minimum for content transfer.
  • USB 3.0 recommended for faster media uploads.
  • Supports USB flash drives, keyboards, and input devices.

Supported Resolutions

IQ World supports a range of display resolutions to accommodate different screen sizes and aspect ratios.

  • 1366 x 768 - standard HD-ready displays.
  • 1920 x 1080 - full HD for crisp, detailed content.
  • 3840 x 2160 - 4K UHD for premium visual experiences.

Performance Recommendations

  • Use a dedicated device for signage to avoid resource contention.
  • Keep the operating system and IQ World software updated.
  • Regularly clear cached content to free storage space.
  • Configure network bandwidth to prioritise signage traffic.
  • Monitor device temperature in enclosed or outdoor installations.

Things To Consider

Review the installation environment, content format, network reliability, and long-term maintenance plan before deployment. Stable hardware and connectivity improve uptime, remote support, and content playback consistency.

4K Playback Considerations

  • 4K content requires higher RAM, 2 GB or more, for smooth playback.
  • Use solid-state storage such as eMMC or SSD for faster media loading.
  • Network speed of 25 Mbps or higher is recommended for 4K streaming.
  • Encode 4K video in H.265 for better compression and performance.
  • Test playback in the target resolution before full deployment.

Outdoor Signage Deployment Guidance

  • Use weather-resistant enclosures for outdoor installations.
  • Ensure adequate ventilation to prevent overheating in direct sunlight.
  • Consider high-brightness displays for outdoor visibility.
  • Secure network connections with weatherproof cabling and connectors.
  • Schedule remote monitoring for early detection of hardware issues.

Hardware Scaling Recommendations

  • Start with minimum specifications and scale based on content complexity.
  • Deploy higher-spec devices for 4K, interactive, or multi-zone layouts.
  • Use consistent hardware across deployments for simplified management.
  • Plan for storage expansion as media libraries grow over time.
  • Evaluate network infrastructure before scaled multi-screen deployments.
